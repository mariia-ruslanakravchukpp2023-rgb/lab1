# VolunTrack — MVP для логістики волонтерських зборів

## Про проєкт
**VolunTrack** — це веб-платформа, розроблена для прозорого управління невеликими волонтерськими зборами. Проєкт допомагає волонтерам звітувати перед донорами, а благодійникам — бачити реальний прогрес та результати допомоги.

### Основні функції (MVP):
* **Моніторинг зборів:** Відстеження накопиченої суми у реальному часі.
* **Прогрес-бари:** Візуалізація того, скільки ще залишилося зібрати до мети.
* **Модуль звітності:** Можливість додавати фото чеків та звіти про передачу допомоги.
* **Аналітика донатів:** Перегляд активності зборів за певний період.

## Технологічний стек
- **Frontend:** React (або Vue) + Vite
- **Стилізація:** CSS-modules / Tailwind
- **Аналітика:** PostHog (для відстеження поведінки користувачів)
- **Моніторинг:** Sentry (для логування технічних помилок)
